# Vite

https://vitejs.dev/

## Scaffolding Your First Vite Project

```bash
$ yarn create @vitejs/app

# use template
yarn create @vitejs/app my-vue-app --template vue
```

### Support template presets

- vanilla
- vanilla-ts
- vue
- vue-ts
- react
- react-ts
- preact
- preact-ts
- lit-element
- lit-element-ts
- svelte
- svelte-ts
